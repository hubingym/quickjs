# quickjs
quickjs for v, 参考[quickjs](https://github.com/horhof/quickjs)  
使用v给quickjs写c扩展  
参考qjs/src/qjs.v和qjs/src/fib.v

## Usage

将quickjs克隆到v模块目录下(.vmodules)  
git clone https://github.com/hubingym/quickjs  ==>  ~/.vmodules/quickjs  

切换到qjs目录  
cd qjs  

编译qjs  
make  

运行qjs  
./qjs.exe js/hello.js
